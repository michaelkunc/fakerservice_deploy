# Fakerservice Deploy

Ansible playbook for deploying www.fakerservice.com. Playbook assumes a user with sudo rights. 

### Infrastructure
- Ubuntu 16.04
- Gunicorn
- Nginx

### [Application](https://github.com/michaelkunc/fakerservice)
- Python 3.5
- Flask
- [Faker](https://github.com/joke2k/faker)
- Travis CI
